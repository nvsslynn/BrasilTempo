<h1>Brasil<span style="color:green;">Tempo</span> 🌦️</h1>

<p>BrasilTempo é uma aplicação web desenvolvida com Python e Flask que consome uma API de clima para exibir informações meteorológicas em tempo real de cidades brasileiras.</p>
<p>O projeto procura demonstrar a integração entre back-end e front-end, consumo de API externa e renderização dinâmica de dados.</p>

<h2>Funcionalidades</h2>
<p>A aplicação informa, com base na API:</p>
<ul>
    <li>Consultar clima por cidade</li>
    <li>Temperaturas</li>
    <li>Condição climática</li>
</ul>

<h2>Tecnologias utilizadas</h2>
<h3>Back-end</h3>
<p>No back-end, o Python atua juntamente com o Flask para criação do servidor web e utiliza o módulo requests para consumir os dados da API externa. Os dados são processados e enviados para o front-end.</p>
<h3>Front-end</h3>
<p>Por outro lado, no front-end, o Jinja integrado ao Flask atua para o processamento dos dados recebidos do back-end e o Bootstrap auxilia para o estilo do site.</p>

<h2>Como eu posso testar?</h2>
<p>Primeiro de tudo, tenha o Python 3.9.13 e lembre-se de configurar o pip no caso de estar utilizando um sistema Linux.</p>
<p>Use o pip no seu terminal/cmd para instalar os requisitos digitando:</p>
<span style="color:green;font-style:italic;">- python3 -m pip install -r requirements.txt</span><br><br>
<p>Após a instalação, crie uma chave de API <a href="console.hgbrasil.com">aqui</a> e coloque na variável `key` do arquivo main. Após isso, basta rodar o arquivo main.py na raiz utilizando o comando:</p>
<span style="color:green;font-style:italic;">- python3 main.py</span>
